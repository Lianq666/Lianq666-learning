# 每日任务复盘面板

一个使用 PowerShell 和 WPF 制作的本地每日任务复盘面板。

它支持：

- 今日任务、作息和完成状态记录
- 今日、本周、本月统计
- 能量、提醒、关键产出和复盘记录
- 本地 JSON 数据保存
- 可选的 Windows 定时任务自动启动
- 深色专注风格界面和 DPI 缩放

## 安全范围

本仓库只包含程序代码、界面资源、测试截图和示例数据。

真实的每日记录不要提交到 GitHub。`data/*.json` 已加入忽略规则；请只保留 `data/example.json` 作为演示。

## 使用方法

1. 使用 Windows PowerShell 5.1 或更高版本。
2. 将仓库下载到本地目录。
3. 直接运行 `Launch-DailyTaskReview.cmd`，或运行 `DailyTaskReview.ps1`。
4. 如需每天自动启动，再运行 `Install-DailyTaskReview.ps1`。
5. 如需取消自动启动，运行 `Uninstall-DailyTaskReview.ps1`。

安装脚本会在当前用户范围内创建一个 Windows 计划任务，并在桌面创建快捷方式。使用前请先阅读脚本内容，确认你了解这些系统变更。

### 安装脚本的系统影响

- 计划任务名称为 `每日任务复盘-07点`，默认每天 07:00 启动面板。
- 快捷方式会创建在当前用户的桌面目录中。
- 安装脚本使用 PowerShell 的 `ExecutionPolicy Bypass` 运行本项目脚本；这是为了避免本地执行策略阻止脚本，不代表脚本应被盲目信任。
- 卸载脚本只处理本项目创建的计划任务和快捷方式，历史 `data` 记录不会自动删除。
- 发布前检查未发现本项目包含网络上传逻辑；程序数据默认只写入本地 `data` 目录。

请在运行安装脚本前检查脚本内容，并确认计划任务、快捷方式和本地数据保存方式符合你的预期。

## 数据位置

程序会在项目目录下创建 `data` 文件夹，并按日期保存 JSON 记录。公开仓库中的 `data/example.json` 仅用于演示格式，不包含真实个人信息。

## 文件说明

- `DailyTaskReview.ps1`：主程序入口
- `MidnightFocusUI.ps1`、`MidnightFocusWPF.ps1`：界面和交互组件
- `Install-DailyTaskReview.ps1`：可选的每日自动启动安装脚本
- `Uninstall-DailyTaskReview.ps1`：移除本项目创建的定时任务和快捷方式
- `assets/`：界面图片资源
- `tests/`：界面检查截图
- `config.json`：默认显示和定时任务配置
